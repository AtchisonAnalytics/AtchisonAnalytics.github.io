Debt Fund Loan Line Analysis - (2nd subset dataset)

BACKGROUND

The below analysis is prepared based on an Excel spreadsheet matrix provided by Alceon focusing on loan line investments made in the context of the Alceon Debt Fund investment manadate. The below is intended to provide an example of the type of data visualisation analysis we are able to prepare in support of both internal portfolio management processes and external marketing requirements.

Per my covering email - please note a number of data columns were not able to be mapped accross in this 2nd data set - so columns such as "Pipeline Status", "Drawdown Type", "Expected Maturity", "State", "AGPLResponsible" were not available - dummy data has been included for now for these columns.

This analysis is prepared in a Python model we have coded internally over the last 18 months. We are able to create the charts and output the analysis to stand alone HTML files (that can be hosted on internal or external websites) for sharing purposes.

All charts can be interacted with (i.e scroll, zoom, filter on and off legend items ect)
